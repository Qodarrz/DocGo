# DocGo Application

Welcome to the DocGo repository. This monorepo contains the source code for the DocGo platform, featuring a Node.js backend, Next.js frontend, and Flutter mobile application.

## Project Structure

The repository is organized into the following main directories:

-   **[backend](./backend)**: The server-side application built with Node.js, Express, and Prisma. Handles API requests, database interactions, and real-time sockets.
-   **[frontend](./frontend)**: The web interface built with Next.js 16 and Tailwind CSS. Provides the administrative and user web dashboard.
-   **[mobile](./mobile)**: The mobile application built with Flutter. Allows users to access DocGo services on Android and iOS devices.

## Getting Started

To get started with development, please refer to the `README.md` files in each individual project directory for specific installation and running instructions.

## Technologies Used

-   **Backend**: Node.js, Express, PostgreSQL, Socket.io, Redis, BullMQ
-   **Frontend**: Next.js, React, Tailwind CSS, Radix UI
-   **Mobile**: Flutter, Dart, GoRouter, Provider, Firebase Messaging, Workmanager, Android Alarm Manager Plus
